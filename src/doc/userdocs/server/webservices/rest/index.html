  1. Introduction

    The Fedora REST API is a new, experimental feature that exposes a subset of the Fedora Access and Management APIs with a REST interface. As the Fedora REST API is an experimental interface, the REST API should not be considered stable. In particular, future releases may implement a different subset of the Fedora Access and Management APIs and may employ different URL endpoints. Moreover, the REST API has not undergone security testing, so production repositories are cautioned against enabling the REST API without thorough evaluation.
